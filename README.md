# Handwritten Digit Recognizer

A neural network built from scratch using only NumPy and CuPy. The network is trained on the MNIST dataset, and achieved 97.32% test accuracy.

## Live Demo

[https://drawadigit.onrender.com](https://drawadigit.onrender.com)

## What it does

Draw a digit (0–9) in the browser. The network classifies it in real time.

## How it works

- **Architecture:** 784 → 128 → 64 → 10
- **Activation:** ReLU on hidden layers, Softmax on output
- **Loss:** Categorical cross entropy
- **Training:** Mini-batch gradient descent with manual backpropagation
- **Dataset:** MNIST (60,000 training images, 10,000 test images)

Everything — forward pass, backpropagation, weight updates — is implemented manually without ML libraries.

## Stack

- Python, NumPy, CuPy
- FastAPI backend
- Docker
- Vanilla HTML/CSS/JS frontend - **GENERATED BY AI**

## Run locally

```bash
docker build -t mnist-app .
docker run -p 8000:8000 mnist-app
```

Then open `http://localhost:8000`
